# 🤖 Personal Assistant Automation Bot

## 🧠 Problem

Performing repetitive tasks on a system such as sending messages, opening applications, or searching content manually can be time-consuming and inefficient.

There is a need for a simple assistant that can execute these actions based on user commands while maintaining control and safety.

---

## 🚀 Solution

Built an automation assistant that executes system-level tasks based on user commands, including messaging, application control, and browser actions.

The assistant ensures safe execution by requiring user approval before performing critical actions.

---

## ⚙️ How It Works

1. **Command Input**
   - User provides a command (e.g., send message, open YouTube, open WhatsApp)

2. **Command Processing**
   - System interprets the intent of the command

3. **Action Execution**
   - Performs tasks such as:
     - Opening WhatsApp Web
     - Searching and selecting contacts
     - Typing messages
     - Opening websites (YouTube, etc.)

4. **Approval Layer (Key Feature)**
   - Before sending messages, the system asks for user confirmation
   - Ensures controlled and safe automation

5. **Contact Mapping**
   - Supports predefined contact directory for sending messages directly using names

---

## 📊 Features

- WhatsApp message automation (with approval)
- Contact-based messaging system
- YouTube and browser automation
- Application control via commands
- Safe execution with confirmation layer
